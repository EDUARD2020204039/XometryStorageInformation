Xometry Analysis - Documentație

Aplicație completă pentru gestionarea și analiza ofertelor de la platforma Xometry.

🌐 Aplicație Web

Backend FastAPI cu interfață web modernă pentru gestionarea ofertelor și reperelor.

  • Dashboard centralizat
  • Editare inline a prețurilor
  • Export CSV/XLSX
  • Gestionare imagini
🔧 Extensie Chrome

Extensie pentru extragerea automată a datelor de pe paginile Xometry.

  • Extragere automată
  • Calculator prețuri
  • Sincronizare timp real
  • Interfață intuitivă

📊 Gestionare Oferte (Jobs/RFQ)
  • Analiză automată a reperelor
  • Prețuri și discount-uri

  • Backup și migrare

Instalare Rapidă

1. Cerințe de Sistem

  • Python 3.8+
  • Chrome Browser

2. Instalare Extensie Chrome

📥 Descarcă Extensia: Descarcă Ultima Versiune (v2.61)
  1. Descarcă arhiva extensiei
  2. Mergi la chrome://extensions/
  3. Activează "Developer mode"
  4. "Load unpacked" -> Selectează folderul extras

API Endpoints

Gestionarea Ofertelor

GET /api/offers - Lista ofertelor
POST /api/scrape - Salvare date din extensie

Gestionarea Comenzilor (Orders)

POST /api/orders/sync - Sincronizare comenzi (Listă sau Detaliu)
GET /api/orders/check/{part_id} - Verificare istoric producție

Utilizare

1. Analiza Ofertelor (Job Board)

  1. Mergi pe o ofertă (Job/RFQ).
  2. Apasă butonul albastru "Analiza" (dreapta-jos).
  3. Se deschide tabelul de calcul. Completează prețurile.
  4. Apasă "Scrapează & Trimite" pentru a salva în backend.

2. Sincronizarea Comenzilor (Orders)

Poți sincroniza comenzile în două moduri:

  • Din Lista de Comenzi: Mergi la pagina "My Orders". Apasă "Sync Orders". Se vor salva sumarul comenzilor.
  • Din Detaliile Comenzii (PO-xxxx): Intră pe o comandă specifică (unde sunt listate reperele). Apasă "Sync Orders". Se vor salva toate detaliile reperelor (dimensiuni, materiale, prețuri per reper, imagini).
