Troubleshooting
Aplicația nu pornește

Cauze posibile:

  • Portul 10000 este ocupat
  • Python nu este instalat corect
  • Dependențele lipsesc

Soluții:

# Verifică portul netstat -tulpn | grep :10000 # Reinstalează dependențele pip install -r requirements.txt # Pornește aplicația python app.py
Extensia nu se conectează

Cauze posibile:

  • URL-ul backend-ului este greșit
  • Aplicația web nu rulează
  • Probleme de CORS

Soluții:

  • Verifică URL-ul în extensie: http://localhost:10000
  • Verifică că aplicația web rulează
  • Verifică log-urile din console
Datele se salvează corect

Dacă totul funcționează, vei vedea:

  • Popup verde în extensie
  • Oferta apare în lista principală
  • Datele sunt complete și corecte
